# Memo

메모를 작성하고 저장합니다. 메모에는 사진 저장이 가능하며 저장한 메모의 수정 및 삭제 또한 가능합니다. 

저장한 메모들은 메모 리스트에서 볼 수 있습니다.

## About the Memo project

평상시에 많이 사용하는 메모 어플의 주요 기능을 담은 웹페이지를 만들어보았습니다.
