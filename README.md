# 📝Memo

메모를 작성하고 저장합니다. 

## About the Memo project

### 개요

평상시에 많이 사용하는 메모 어플의 주요 기능을 담은 웹페이지를 만들어보았습니다.

### 일정

개인 프로젝트
<br>2023년 2월

### Stacks

```
Java Eclipse Spring Boot MyBatis

Javascript HTML CSS

MySQL GitHub Zenhub
```

## 기능 구현

* 메모를 작성합니다. 메모에는 사진 첨부가 가능하며 저장한 메모의 수정 및 삭제가 가능합니다.
* 메모 리스트에서 저장한 메모들을 볼 수 있습니다.
